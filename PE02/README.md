# 1. Input 
- User input for filtering movies by genre. 
# 2. Process
- Create a MovieList component that will be the main component of the application.
- Receive user input for filtering movies by genre and update the state of the selected genre.
- Create a filtered list of movies based on the selected genre and the default list of movies.
- Render the list of movies, of which each movie should be displayed as a card with its title, genre, and release year.
# 3. Output
- All movies are displayed as first when the page is loaded.
- When the user selects a genre from the dropdown menu, only movies of that genre are displayed.
- When the user click on one of the movies, an alert box will pop up showing the title of the movie.
- CSS styles are applied to the movie card components to enhance the visual appearance of the movie list.
